# Reservas para restaurante

Una aplicación web para gestionar reservas de restaurantes.

## Características
- Reservas de mesas por fecha y hora
- Gestión de capacidad (2, 3, 4+ personas)
- Frontend en Preact con Tailwind CSS (Vercel)
- Backend con Node.js/Express (Render)
- Base de datos PostgreSQL (Supabase)
- Gestión de estado con Zustand
- Gestionamiento de correos electrónicos (Resend)

## Tecnologías
- Preact
- Tailwind CSS
- Zustand
- Node.js
- Express
- PostgreSQL
